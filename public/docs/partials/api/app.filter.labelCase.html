labelCase
filter in module app

Description

This filter:

  1. Adds a space before every upper case letter.
  2. Capitalizes the first letter of the string.

Usage

In HTML Template Binding

{{ labelCase_expression | labelCase }}

In JavaScript

$filter('labelCase')(input)

Parameters

ParamTypeDetails
inputString

String to be transformed to labelCase

Returns

String

String which has been transformed to labelCase
