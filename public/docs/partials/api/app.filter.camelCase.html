camelCase
filter in module app

Description

This filter:

  1. Converts all letters to lowercase.
  2. Wherever a space is found, the letter following the space is capitalized, and the space is removed.

Usage

In HTML Template Binding

{{ camelCase_expression | camelCase }}

In JavaScript

$filter('camelCase')(input)

Parameters

ParamTypeDetails
inputString

String to be transformed to camelCase

Returns

String

String which has been transformed to camelCase
