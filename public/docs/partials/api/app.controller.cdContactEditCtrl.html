cdContactEditCtrl
controller in module app

Description

Controller which handles the editing of a contact's details.

Dependencies

  • $scope
  • contact
  • cdNotifierSvc
  • cdGoogleMapsSvc
  • cdGoogleMapsCoordinatesSvc
  • contactFieldsSvc
  • contactAddressFieldsSvc

Methods

  • $viewContentLoaded()

    This method handles intitializing the Google Map.

    • Wait until view content is loaded so Google Maps will have access to the #map element.
    • Check to see if contact's address is defined
      • If not, initialze Google Map with no specific address
      • If so, pass the address array to
        1. Pass the address array to cdGoogleMapsCoordinatesSvc to obtain the addresses coordinates.
        2. Pass returned coordinates to cdGoogleMapsSvc to initiaclize the map with a location pin and zoomed to that pin.

  • saveContact()

    This method will redirect page to the /contact/add page.

    1. Contact Update
      1. ng-Model used to bind input values to contact object values.
      2. Call cdContactSvc.$update() method to update the contact's information in the database.
      3. Notify user the contact was updated successfully via cdNotifierSvc (toastr notification).
    2. Google Maps
      • Check to see if contact's address is defined
        • If not, initialze Google Map with no specific address
        • If so, pass the address array to
          1. Pass the address array to cdGoogleMapsCoordinatesSvc to obtain the addresses coordinates.
          2. Pass returned coordinates to cdGoogleMapsSvc to initiaclize the map with a location pin and zoomed to that pin.

Properties

  • $scope.addressFields

    Variable containing all contact's address field information defined in contactAddressFieldsSvc.

  • $scope.contact

    Variable containing contact object which was created by the $routeProvider in app.js.

  • $scope.fields

    Variable containing all contacts field information defined in contactFieldsSvc.
