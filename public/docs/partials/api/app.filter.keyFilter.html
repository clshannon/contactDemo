keyFilter
filter in module app

Description

This filter:

  1. Create empty result object.
  2. Iterate through key/value pairs in object.
    • If key doesn't match query
      • Add key/value pair to result object
    • If key does match query
      • Key/value pair is not added to result object thus omitting it from the returned object.

Usage

In HTML Template Binding

{{ keyFilter_expression | keyFilter }}

In JavaScript

$filter('keyFilter')(obj)

Parameters

ParamTypeDetails
objObject

Object containing keys and values to be filtered

Returns

Object

Object containing remaining key/value pairs
