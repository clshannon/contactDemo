cdContactAddCtrl
controller in module app

Description

Controller which handles adding a contact to the database.

Dependencies

  • $scope
  • $location
  • cdContactSvc
  • cdNotifierSvc
  • contactFieldsSvc
  • contactAddressFieldsSvc

Methods

  • addContact()

    This method will redirect page to the /contact/add page.

    1. Call cdContactSvc.$update() method to add contact to database
      • If successful:
        1. Notify user of success via cdNotifierSvc (toastr notification).
        2. Redirect page to /contact/:id based on new user's returned _id

Properties

  • $scope.addressFields

    Variable containing all contact's address field metadata defined in contactAddressFieldsSvc.

  • $scope.contact

    Variable containing contact object initialized from cdContactSvc.

  • $scope.fields

    Variable containing all contacts field metadata defined in contactFieldsSvc.
