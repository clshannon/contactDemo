formField
directive in module app

Description

This directive:

  1. Creates an html template for form inputs using bootstrap markup.
  2. Passes FieldTypes value to template to create ng-message messages for form input errors.

Usage

as element:
<form-field
       record="{Object}"
       field="{String}"
       live="{String}"
       required
       type="{String}">
</form-field>
as attribute
<form-input form-field
     record="{Object}"
     field="{String}"
     live="{String}"
     required
     type="{String}">
   ...
</form-input>

Directive info

  • This directive creates new scope.
